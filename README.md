# AI Agents in Practice — Companion Code

*Companion code for the [AI Agents in Practice](https://dev.to/gursharansingh/series/39810) article series, part of
[AI in Practice](https://aiinpracticehub.com) — a portfolio on production-ready AI engineering.*

This repo contains article-specific, runnable labs (such as `part6/`). Examples are
deterministic and require no API keys.
